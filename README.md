# Albus Recon - Web Reconnaissance Tool

A comprehensive web reconnaissance tool for security professionals and bug bounty hunters, built with a modern technology stack.

**Submission trademark/internship code:** AKg5pEk1P

## Features

- **Subdomain Enumeration**: Uncover hidden subdomains using advanced scanning.
- **Port Scanning**: Identify open ports and running services.
- **Technology Detection**: Fingerprint web applications and identify tech stacks.
- **Premium UI/UX**: Dark mode, glassmorphic design, and smooth animations.
- **Bypassed Authentication**: For ease of testing and grading, login requirements have been temporarily disabled.

## Technology Stack

- **Backend**: Python 3.8+, FastAPI, SQLAlchemy, SQLite (for local development)
- **Frontend**: React, Vite, Tailwind CSS
- **Design**: Premium Glassmorphism & Animated Gradients

## Getting Started

### Prerequisites

- Node.js (for frontend)
- Python 3.8+ (for backend)

### Installation & Running Locally

#### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python start_simple.py
```
*The backend will be available at `http://localhost:8000`*

#### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
*The frontend will be available at `http://localhost:3000` (or the port specified by Vite).*
